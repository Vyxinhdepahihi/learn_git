# Các lệnh git cần nhớ

1. git init 
    - Khởi tạo nơi lưu
2. git add
    - Thêm file thay đổi trước khi commit
3. git commit
    - Lưu trạng thái thành 1 bược
4. git push
    - Dùng để đẩy lên remote
5. git pull
    - Kéo từ remote về local
